Instructions:
------------------------------------------------------------------------
  Left click: Place cat

  Right click: Place fish

  WASD/Arrows/Mouse: Move Camera

  +/- / MouseWheel: Zoom Camera

  M key + 1-5: Change Map

  V key: show/hide pathfinding

  P key: Pause

  Esc key: Quit
